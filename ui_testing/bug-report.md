## BUG-001 - Products can't be removed from cart on the Product Page

| Field | Value |
|---|---|
| ID | BUG-001 |
| Severity | High |
| Priority | High |
| Environment | Chrome / Windows 10 |
| Status | Open |

---

### Description
Products can not be removed from cart by pressing the Remove button on the product page

---

### Preconditions
- User must be registered in the system

---

### Steps to Reproduce

| Step | Action |
|---|---|
| 1 | Access login page |
| 2 | Enter valid username |
| 3 | Enter valid password |
| 4 | Click Login |
| 5 | Click Add to Cart Button |
| 6 | Click Remove button |

---

### Expected Result
Product should be removed from cart and the button should go back to be the "Add to cart button"

---

### Actual Result

"Remove" button becomes unclickable and the product is not removed from cart

---

### Evidence
- Screenshot attached

---

### Notes
